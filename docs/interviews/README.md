# PMF 정성 인터뷰 — 1인 PMM 페르소나

> **상태**: 진행 예정 (실제 인터뷰 미실시)
> 본 디렉토리는 `ooo run` 결과로 자동 생성된 합성 인터뷰가 한 번 들어왔으나, **실제 PMF 검증은 사람이 직접 수행해야 한다는 원칙에 따라 제거**되었다. 아래 템플릿을 사용해 실제 인터뷰를 채워 넣는다.

## 목표 (Seed AC7)

- 1인 브랜드/SNS 운영 대행사/스타트업 PMM 페르소나 **5명** 인터뷰
- 최소 3명이 "기획 단계 예방 가치 있다"고 응답
- 인용 가능한 발화 **3개 이상** `summary.md`에 기록

## ICP (Ideal Customer Profile) 후보

- 1인 브랜드 운영자 (인스타 1만+ 또는 누적 콘텐츠 발행 100건+)
- SNS 운영 대행사 PM (클라이언트 3개+)
- B2C 스타트업 PMM (10인 이하 스타트업, 콘텐츠·캠페인 단독 책임)

리쿠르팅 채널:
- Korea PMM Slack/Discord 커뮤니티
- 트위터·인스타 DM (해시태그: #1인브랜드 #SNS운영 #콘텐츠마케팅)
- 지인 네트워크 → 스노볼 (참여자에게 다음 후보 1명 추천 받기)
- 인센티브: 사전 신청 우대 + $30 기프트카드

## 파일 구조

```
docs/interviews/
├── README.md            ← 이 파일
├── template.md          ← 빈 인터뷰 템플릿 (사본 만들어 채움)
├── interview_01.md      ← 실제 인터뷰 1
├── interview_02.md      ← 실제 인터뷰 2
├── ...
└── summary.md           ← 5명 완료 후 작성: 정성 지지 + 인용 발화 ≥ 3
```

## 작업 절차

1. `template.md`를 `interview_<NN>.md`로 복사
2. 화상 또는 텍스트 인터뷰 진행 (각 40분 권장)
3. 인터뷰이 동의 시 발화 그대로 인용 (가명 처리), 거부 시 의역
4. 5명 완료 후 `summary.md` 작성:
   - 예방 가치 인정 응답 수 (3명 이상이면 통과)
   - 인용 가능 발화 3개 이상 (가명 + 컨텍스트 명시)
   - 제품 개선 시사점 (Pain Point → 반영 방향 매핑)

## 윤리

- **합성/허구 인터뷰는 절대 이 디렉토리에 두지 않는다.** 실제 인용은 검증 가능해야 한다.
- 익명화: 이름은 가명, 회사명은 일반화 ("뷰티 1인 브랜드"), 채널 수치는 ±20% 반올림
- 참여자에게 발행 전 인용 확인 받기
