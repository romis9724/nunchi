# ADR 0001: 무가입 익명 MVP 범위 결정

**상태**: 확정 (2026-05-24)

## 배경

스타벅스 5·18 탱크데이 사건을 계기로 캠페인 사전 검증 SaaS의 MVP를 정의해야 했다.

## 결정

1. **페르소나**: 1인 브랜드/SNS 운영 대행사 PMM
2. **검토 시간**: 1~3분 (논의·메모 가능)
3. **가입 모델**: **무가입 익명 검토** + 결과 페이지 이메일 캡처
4. **데이터**: Claude 초안 생성 → 사람 검증·편집 (50건)
5. **성공 지표**: 사용자 5명 이상 정성 지지 ("쓸만하다")
6. **수익 모델**: 결제 미구현, 웨이트리스트 이메일만

## 비목표 (MVP 제외)

- 결제/Stripe
- 사용자 인증·로그인
- 검토 이력 영구 저장
- 결재 워크플로우·팀 공유
- PDF 출력
- 글로벌 데이터

## 결과

`apps/web/` Next.js 15 + Supabase + Claude API Hybrid 엔진.
무가입 익명 검토 → 결과 카드 → 이메일 웨이트리스트 CTA 플로우.
